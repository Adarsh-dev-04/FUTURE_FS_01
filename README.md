## ✨ Features

- 🌙 **Dark / Light Mode** — Toggle with persistent preference via `localStorage`
- 🖱️ **Custom Cursor** — Smooth animated cursor with hover effects
- 🎞️ **Scroll Reveal Animations** — Sections animate in as you scroll
- 📱 **Fully Responsive** — Works seamlessly on mobile, tablet, and desktop
- 🎨 **Creative Design** — Editorial typography with electric blue accents
- 📬 **Contact Form** — Functional form with validation and success feedback
- ⚡ **No Frameworks** — Pure HTML, CSS & JavaScript — zero dependencies

## 🚀 Getting Started

### View Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/Adarsh-dev-04/portfolio.git
   ```

2. **Navigate into the folder**
   ```bash
   cd portfolio
   ```

3. **Open in browser**
   ```bash
   # Just open index.html directly, or use Live Server in VS Code
   open index.html
   ```

> 💡 Tip: Use the **Live Server** extension in VS Code for the best local development experience.
